# Netiquette Compass

Basit bir React + Vite uygulaması — Dijital nezaket görevleri sunar.

Yerel geliştirme

```bash
npm install
npm run dev
```

GitHub Pages'e deploy

1. `package.json` içinde `homepage` veya `vite.config.js`'de `base` ayarlı olmalı (repo adıyla eşleşir): `/netiquette-compass/`.
2. `gh-pages` yüklü ise aşağıdaki komutla deploy edebilirsiniz:

```bash
npm run deploy
```
